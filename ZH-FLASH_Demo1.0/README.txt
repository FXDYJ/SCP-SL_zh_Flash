## 使用方法

1. 解压压缩包后将名为"zh_Flash_Demo1.0"的文件夹拖拽至游戏根目录中的"Translations"文件夹内.
2. 之后启动游戏在语言选项找到"简体中文-原版重置1.0.0-alpha"应用并重启游戏即可.
